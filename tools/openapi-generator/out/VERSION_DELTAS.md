# WebLogic REST Management API — Cross-Version Deltas

Diffs between adjacent supported WLS versions, computed from the specs generated by `tools/openapi-generator/`. Each pair compares the older release's spec against the next release's spec.

All counts are computed against the **generated** specs, not the harvested source. Generated specs include schemas for the 22 MBeans we curate plus stub schemas for every other MBean reachable from the containment graph; stub schemas are excluded from the *property-level* diff (no body to compare) but their presence or absence is reflected in the path counts and total schema counts.
